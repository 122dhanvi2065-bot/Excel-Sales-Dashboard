# Myntra Sales Data Analysis using Excel

## 📌 Project Overview
This project analyzes Myntra sales data using Microsoft Excel. 
The dataset is explored using pivot tables, charts, and data visualization techniques to generate useful business insights.

## 📊 Dataset
The dataset contains information about Myntra products such as:
- Product category
- Sales data
- Ratings
- Price
- Customer reviews

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts and Graphs
- Data Cleaning Techniques

## 📈 Analysis Performed
The following analyses were conducted:

- Sales distribution by product category
- Product rating analysis
- Price vs rating analysis
- Sales trends
- Customer review insights

## 📉 Visualizations
The project includes:
- Pivot tables
- Bar charts
- Sales analysis charts
- Summary dashboards

## 📂 Project Files
- `Dhanvi_K_Excel_project.xlsx` – Excel workbook containing dataset and analysis
- `README.md` – Project documentation

## 🎯 Key Insights
- Identified top-performing product categories
- Observed relationship between product price and ratings
- Found categories with highest customer engagement
